## Description

Backend server for mu personal stuff. Maybe there is something useful for you here...

So far it serves the purpose of storing and sending IPs I want for ssh connection.

This is written using Nestjs.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
